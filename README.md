# Snake-Game

### Create a virtual enviroment
  ```
  $ conda create -n pygame _env
  ```
  
### Activate the enviroment
  ```
  $ conda activate pygame_env
  ```
  
### Install Pygame
  ```
  $ pip install pygame 
  ```

### Run the game
  ```
  $ python snake.py
  ```
